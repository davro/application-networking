Shared drawing canvas and drag & drop image bucket!

Drawing Canvas: HTML 5 canvas example where muktiple users can draw at the same time and the drawings are shared among the users with NodeJS. HTML 5 localStorage is also used in this app. You can also chat with others users while you draw!

Image Bucket: This is a Drag & Drop image bucket where multiple users can drag images on to the website and these images will be shared with all the users. Users can move the images, delete them and of course download them!
